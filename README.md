## 📌 Project Name

**Digital Tools Buying Website**

---

## 📖 Description

Digital Tools Buying Website is a modern React-based web application where users can explore different digital tools and add them to a cart before purchasing.
The project focuses on building a clean UI, implementing cart functionality, and practicing modern frontend development using React, Tailwind CSS, and DaisyUI.

---

## ⚙️ Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* DaisyUI
* React Icons
* React Toastify
* JSON Data

---

## ✨ Key Features

### 🛒 Product & Cart System

Users can browse digital products and add them to a cart. The cart dynamically updates and displays selected products.

### 🔔 Smart Notifications

React Toastify is used to show alerts when a product is added to cart, removed from cart, or when checkout is completed.

### 🔄 Product & Cart Toggle

Users can easily switch between the **Product section** and **Cart section** using toggle buttons in the interface.
